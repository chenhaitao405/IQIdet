像质计文字标注说明

本压缩包用于像质计文字图片标注。请严格按照下面步骤操作。

1. 解压要求

请先将整个压缩包完整解压到一个独立文件夹中，再开始操作。

注意：

  • 不要在压缩包预览界面里直接运行 exe
  • 不要只解压其中一部分文件
  • exe、待标注图片文件夹、说明文件必须保持在同一个解压目录内

2. 启动程序并选择待标注文件夹

双击打开标注程序 label_rec_no_box.exe。

程序启动后会弹出文件夹选择框：

  • 如果由 1 人独立完成全部标注，请选择大文件夹 iqi_rec_textdet_v1
  • 如果由 多人分工标注，请只选择自己分配到的子文件夹

多人分工示例：

iqi_rec_textdet_v1/det_crops/all/images__bde9ea/1120

也就是说，每位标注人员只处理自己分配到的那个文件夹，不要重复标注其他人的文件夹。

如果程序启动后浏览器没有自动打开，可手动访问：

http://127.0.0.1:8766/

3. 标注规则

  1. 如果图片方向不正，必须先把图片旋转 / 镜像到文字方向正确后，再填写标注内容。原因是后续文本矫正模型需要使用这些数据，输入图像必须保证文字方向正确。
  2. 遇到文字重叠、严重遮挡、完全无法辨识的图片，才允许点击“跳过 => ~”。
  3. 只要文字可以辨识，就禁止跳过，必须填写真实内容。
  4. 十字箭头请统一标注为：+
  5. 所有英文字母请统一使用大写

常用按钮和快捷键：

  • 保存并下一张：Ctrl+E 或回车
  • 跳过 => ~：Ctrl+Q
  • 左转 90 度：A
  • 右转 90 度：D
  • 水平镜像：M
  • 重置预览：R

4. 标注完成后的回传要求

当所选文件夹中的所有图片都标注完成后，程序会在 exe 同级目录自动生成 3 个文件：

  • <所选文件夹名>.txt
  • <所选文件夹名>_labels.tsv
  • <所选文件夹名>_session.json

请将以下内容一起打包成一个压缩包并返回：

  • 上面自动生成的 3 个标注文件
  • 你本次实际选择并标注的那个文件夹

例如，如果你选择并标注的是：

iqi_rec_textdet_v1/det_crops/all/images__bde9ea/1120

则回传压缩包中应至少包含：

  • 1120.txt
  • 1120_labels.tsv
  • 1120_session.json
  • 文件夹 1120

5. 额外提醒

  • 每次建议只处理一个分配好的文件夹
  • 标注过程中不要随意改动图片文件夹结构
  • 如中途退出，下次可继续打开程序并重新选择同一个文件夹，程序会自动接续之前的标注进度
